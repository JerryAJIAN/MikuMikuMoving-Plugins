MikuMikuMoving 向けプラグイン


概要
====

MikuMikuMoving 向けのプラグインです。
以下のものが含まれます。

* AnimateCaptionPlugin			: 字幕アニメーション
* ApplyNoisePlugin				: ノイズ付加
* ApplyOffsetPlugin				: オフセット付加
* ApplyRotationPlugin			: 任意中心回転付加
* BeatBasedCopyPlugin			: ビートコピー
* DetailedStatusPlugin			: 詳細情報表示
* DetailedTimePlugin			: 詳細時間表示
* GetMmdTransformationPlugin	: MMD ポーズ取得
* MotionUtil					: 各プラグイン共通ファイル
* RevertFramesPlugin			: フレーム時間反転
* SetMmdTransformationPlugin	: MMD ポーズ設定
